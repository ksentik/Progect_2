## Краткое описание
Консольная программа для учёта ежедневных расходов за месяц. Пользователь может добавлять расходы (день, сумма, категория), просматривать аналитику за любой период, отменять последнее действие и просматривать категории в дереве.

## Возможности программы
1. **Добавление расхода** - день (1–31), сумма, категория
2. **Сумма за период** - выводит сумму расходов с дня A по день B
3. **День с максимальными расходами** - находит день, в который потрачено больше всего
4. **Категории по сумме** - выводит все категории, отсортированные по убыванию суммы
5. **Отмена последнего действия** - удаляет последний добавленный расход
6. **Просмотр дерева категорий** - выводит категории в алфавитном порядке
